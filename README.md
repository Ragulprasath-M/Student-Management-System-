# Student Management System

This is a simple Student Management System project developed using HTML, CSS, and JavaScript. The main goal of this project is to manage student records in an easy and organized way through a clean web interface.

In this application, users can add new student details, edit existing records, and delete student information when needed. The data is stored in the browser using Local Storage, so the records remain saved even after refreshing the page.

## Features

- Add student details
- Edit student information
- Delete student records
- Display student list dynamically
- Data stored using browser Local Storage
- Responsive design for different screen sizes

## Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)

## Purpose of the Project

I built this project to improve my frontend development skills, especially in DOM manipulation, form handling, and JavaScript logic building. This project helped me understand how real-world applications manage and display data dynamically.

## How to Run

1. Download or clone the repository.
2. Open the `index.html` file in your browser.
3. Start adding and managing student records.
